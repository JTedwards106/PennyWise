# Extension Icons

You need to create 4 PNG icon files in this directory:

- `icon16.png` (16x16 pixels)
- `icon32.png` (32x32 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

## Quick Options

### Option 1: Online Generator (Easiest)
1. Visit https://www.favicon-generator.org/
2. Upload any shopping-related image
3. Download generated icons
4. Rename and place here

### Option 2: Free Icon Sets
- Search "shopping cart icon png" on Google Images
- Use sites like flaticon.com or iconfinder.com
- Download in multiple sizes

### Option 3: Create Your Own
Use any image editor (Paint, Photoshop, GIMP, Canva, etc.)

## Design Suggestions

**Good Icon Ideas:**
- Shopping cart 🛒
- Shopping bag 🛍️
- Price tag 🏷️
- Coin/money 💰
- Piggy bank 🐷
- Discount/percentage symbol

**Design Tips:**
- Keep it simple - complex details won't show at small sizes
- Use high contrast colors
- Make it recognizable even at 16x16
- Consider using green (matches PennyWise brand color)

## Quick Test Icons

For testing, you can even use:
- Simple colored squares
- Letters "PW" on a background
- Any placeholder image

The extension will work without perfect icons during development!
